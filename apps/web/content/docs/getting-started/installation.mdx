# Installation

Structure Creator is available for macOS. Download the latest version from GitHub Releases.

## Download

1. Go to the [Releases page](https://github.com/eddiemoore/structurecreator/releases)
2. Download the latest `.dmg` file for your Mac:
   - `Structure-Creator_x.x.x_x64.dmg` for Intel Macs
   - `Structure-Creator_x.x.x_aarch64.dmg` for Apple Silicon Macs

## Install

1. Open the downloaded `.dmg` file
2. Drag **Structure Creator** to your **Applications** folder
3. Open the app from Applications

### First Launch

On first launch, you may see a security warning since the app is not signed with an Apple Developer certificate.

To open the app:

1. Right-click (or Control-click) on the app in Finder
2. Select **Open** from the context menu
3. Click **Open** in the dialog that appears

This only needs to be done once. After that, you can open the app normally.

## Updates

Structure Creator will notify you when updates are available. You can also check for updates manually:

1. Open Structure Creator
2. Go to **Structure Creator > Check for Updates** in the menu bar

## Uninstall

To uninstall Structure Creator:

1. Quit the application if it's running
2. Drag **Structure Creator** from Applications to the Trash
3. Optionally, remove app data:
   - `~/Library/Application Support/com.structurecreator.app/`
