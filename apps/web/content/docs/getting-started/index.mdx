# Getting Started

Structure Creator is a desktop application for generating folder and file structures from XML schemas. It's perfect for project scaffolding, boilerplate generation, and maintaining consistent project structures.

## Features

- **XML Schema Definition** - Define your folder and file structures using intuitive XML syntax
- **Variable Substitution** - Use `%VARIABLE%` syntax with powerful transformations
- **Template Library** - Save and reuse your schemas as templates
- **Template Wizard** - Create interactive configuration wizards
- **Live Preview** - See your structure before creating it
- **Watch Mode** - Automatically recreate on schema changes
